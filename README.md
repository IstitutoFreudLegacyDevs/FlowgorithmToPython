# **LD ChatBot**
**LD ChatBot** è un'intelligenza artificiale creata dal team Legacy. È progettata per interagire con gli utenti in modo naturale tramite chat.

## **Caratteristiche**
- Risposte rapide e precise.
- Comprensione del linguaggio naturale.
